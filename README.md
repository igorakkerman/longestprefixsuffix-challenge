# Longest Prefix/Suffix (LPS)

### Task
Given a string _s_, find the longest proper prefix of _s_ that is also a proper suffix of _s_.
A _proper_ prefix/suffix is not equal to _s_ itself.

### Examples
#### Example 1
_s = "ABABABA"_  
_Result: length("ABABA") = 5_

#### Example 2
_s = "ABAABB"_  
_Result: length("") = 0_
